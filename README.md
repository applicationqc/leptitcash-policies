# Le P'tit Cash - Politiques et Support

Site web de politiques et support pour l'application mobile "Le P'tit Cash".

## 🌐 Accéder au site web

Votre site est accessible à l'adresse suivante :

### **https://applicationqc.github.io/leptitcash-policies/**

## 📋 Contenu du site

Le site contient :
- **Page d'accueil** (`index.html`) : Page principale avec liens vers les différentes sections
- **Suppression de compte** (`account_deletion.html`) : Page permettant aux utilisateurs de demander la suppression de leur compte
- **Politique de confidentialité** : Lien externe vers la politique de confidentialité

## ⚙️ Activer GitHub Pages (si le site n'est pas accessible)

Si le site n'est pas encore accessible, suivez ces étapes pour activer GitHub Pages :

1. Allez sur votre dépôt GitHub : https://github.com/applicationqc/leptitcash-policies
2. Cliquez sur **Settings** (Paramètres)
3. Dans le menu de gauche, cliquez sur **Pages**
4. Sous "Source", sélectionnez :
   - **Branch** : `main` (ou la branche où se trouvent vos fichiers HTML)
   - **Folder** : `/ (root)`
5. Cliquez sur **Save**
6. Attendez quelques minutes que GitHub déploie votre site
7. L'URL de votre site apparaîtra en haut de la page

## 📱 Utilisation

Ce site est destiné à être utilisé comme page de support pour votre application mobile "Le P'tit Cash". Vous pouvez :

- Partager l'URL avec vos utilisateurs
- L'inclure dans votre application mobile
- L'utiliser pour les soumissions App Store/Google Play (politiques de confidentialité et suppression de compte)
